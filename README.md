# Jogo da memória

## 🎮 Como funciona

O jogo mostra uma sequência de cores. Você precisa repetir a sequência clicando nos botões coloridos. A cada nível, a sequência aumenta. Se errar, o jogo reinicia

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (puro)
